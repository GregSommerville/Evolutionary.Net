This is the source code for the Evolutionary.Net machine learning framework.

How to Use This:
1.  Compile into the Evolutionary.Net.dll, and add a reference to that in your project.  Or use a previously compiled DLL downloaded from the repo.
2.  Add "using Evolutionary" to your code.
3.  Instantiate the engine, and add functions, variables, terminal functions, and constants to the engine's supply of program components.
4.  Write a fitness function that evaluates a candidate and returns a numeric fitness score.
5.  Run the engine and use the resulting genetic program to solve a problem.

Please see the Github repo for more information.
